# 🚚 LogiTrack — MVP Gestión de Envíos

Proyecto académico correspondiente a **Laboratorio de Construcción de Software / Proyecto Profesional I (2026)**.

---

## 📌 Descripción

LogiTrack es un MVP de un sistema de gestión de envíos que permite registrar, consultar y actualizar el estado de envíos dentro de un entorno simulado.

Este proyecto forma parte del desarrollo del **Paquete Base** reutilizable para el TP Principal.

---

## 🎯 Objetivos

- Aplicar Scrum en un proyecto real simulado
- Construir un MVP funcional
- Implementar CI (Integración Continua)
- Diseñar arquitectura base reutilizable
- Incorporar un modelo simple de Machine Learning
- Considerar aspectos éticos y legales (Ley 25.326)

---

## 🧱 Alcance del MVP

✔ Alta de envíos con tracking ID automático  
✔ Listado de envíos  
✔ Búsqueda por tracking ID o destinatario  
✔ Cambio de estado (con auditoría)  
✔ Roles simulados (Operador / Supervisor)  
✔ Priorización de envíos (ML mock)

---

## ❌ Fuera de alcance

- Integraciones reales (couriers, mapas)
- Notificaciones reales (email/SMS)
- Pagos y facturación
- Optimización de rutas
